## Quick Start

### Requirements

- Ruby 3.1.4
- RSpec (optional; only for running tests)

### How to run

```bash
ruby driver.rb
```

### How to test

```bash
# if not installed already
gem install rspec

cd specs
rspec Node.spec.rb
rspec Tree.spec.rb
```

## Misc Notes

### Array slice / subarray using range `..` is not the same as `...` in certain cases e.g.

```ruby
array = [1,2,3,4,5]
array[0..0]         # => [1]
array[0...0]        # => []
```
